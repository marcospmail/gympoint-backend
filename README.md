<h2 align="center">
  Instruções para RocketSeat
</h2>

### 1. Executar
```
yarn install
```

### 2. Criar banco `gympoint` no postgres

### 3. Configurar arquivo .env

### 4. Executar
```
yarn sequelize db:migrate
```
### 5. Executar 
```
yarn sequelize db:seed:all
```
