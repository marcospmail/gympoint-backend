<h2 align="center">
  Instruções para RocketSeat
</h2>

### Executar
``
yarn install
``

### Criar banco `gympoint` no postgres

### Executar
``
yarn sequelize db:migrate
``
### Rodar 
``
yarn sequelize db:seed:all
``
